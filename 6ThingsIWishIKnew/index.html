Key Takeaways

  • look at the design. analyse things find that repeat themselves. helps you plan
  • seperate content form layout. layouts repeat themselves a lot so be smart
  • circle back to the fundamentals because you maybe did not get it the first time
  • take designs that already exsist and go from there
  • do not strive for pixel perfection. don't try to make everything exactly as the plan or perfect on every browser. Catch the essense of the design
  • DOM manipulation helps you learn JS
  • try to build something and you will find the things you need along the way
