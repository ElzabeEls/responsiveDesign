Key Takeaways

  • The srcset attribute allows providing different-sized versions of the same image, letting the browser choose the appropriate version.
  • To change the image completely, use the picture element, which builds upon the img element.
  • The picture element wraps around an img element, and it won't work without an img element nested inside it.
  • Like srcset, the picture element updates the src attribute in the img element but provides commands instead of suggestions, giving more control.
  • Inside a picture element, multiple source elements can be specified, each with its own srcset attribute.
  • The browser executes the first source element it can use.
  • Example with three different images in different formats (AVIF, WebP, and JPEG) demonstrates using new formats without sacrificing backward compatibility.
  • The type attribute in the source element specifies the kind of image format.
