Key Takeaways

  • The biggest difference between using a picture element and not using a picture element is that a picture element commands the browser what to do while not using a picture element only gives suggestions.
  • You can use various source tags within a picture element along with srcset to for example use newer image formats if the browser is capable while keeping the default .jpg image in the img tag in the picture tag as a fall back.
  • You can use various source tags along with different media queries in the media attribute so use different image sizes
