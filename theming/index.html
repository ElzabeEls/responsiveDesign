Key Takeaway

  • <meta name="theme-color" content="#00D494" />
  • You can update the value of theme-color using JavaScript.
  • You can also specify a theme color in a web app manifest file that you link in the head
  • If a visitor decides to add your website to their home screen, the browser will use the information in your manifest file to display an appropriate shortcut.
  • use prefers-color-scheme for light and dark mode: @media (prefers-color-scheme: dark) { // Styles for a dark theme. }
  • <meta name="theme-color" content="#ffffff" media="(prefers-color-scheme: light)" />
    <meta name="theme-color" content="#000000" media="(prefers-color-scheme: dark)" />
  • SVG favicons for dark mode icons.
  • CSS custom properties
    html { --page-color: white; --ink-color: black; } @media (prefers-color-scheme: dark) { html { --page-color: black; --ink-color: white; } }
  • building a color scheme
  • use a CSS filter to dim photographic images in dark mode if you want: filter: brightness(.8) contrast(1.2);
  • might have to swop out images in dark mode
  • include theme info in CSS to let the browser know not to use the default color palette for form fields. Or add to the head: <meta name="supported-color-schemes" content="light dark">
  • Use the accent-color property in CSS to style checkboxes, radio buttons, and some other form fields.
  • more on tinting.
