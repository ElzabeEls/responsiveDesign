Container Queries

There's an upcoming CSS technology that reports the dimensions of any parent container: container queries. Container queries allow you to style components in an independent way. The width of the viewport is no longer what matters. You can write rules based on the width of the containing element. Container queries are a game-changer for micro layouts. Your components can be self-contained, independent of the browser viewport.

Caution: Container queries are a new experimental technology that isn't widely available in browsers. To test the code below, and see the example working, enable container queries in Chrome. Go to chrome://flags/, search for Container Queries, and enable the #enable-container-queries flag. With the flag enabled, you can inspect and debug container queries in Chrome DevTools.

You can use media queries for the page layout, and container queries for the components within the page.
